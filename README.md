# FishingMap

Sovelluksen tarkoituksena on merkitä muistiin parhaat kalapaikat. Sovellus avautuu karttanäkymään ja hakee sijainnin. 
Karttanäkymässä on nappi saaliin merkitsemistä varten. Tekstikenttään merkitään saalis lisätietoineen. Teksti 
lähetetään Firebaseen, tekstin mukana lähetetään sijainti. Sovellus on todella pahasti kesken, eikä todennäköisesti 
toimi lainkaan tällä hetkellä. En ole saanut expoa toimimaan pariin päivään, joten kehitys on tapahtunut sokkona. 
Mikäli aikaa jää ja saan sovelluksen toimimaan on tarkoituksena lisätä ominaisuus firebasen tietojen lukuun siten, että saaliiden
sijainnit näytetään kartalla ja saaliin merkkiä klikkaamalla näytetään ylös kirjatut tiedot saaliista.
